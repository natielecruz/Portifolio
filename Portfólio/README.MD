# ![Portfólio](https://img.shields.io/badge/Portfólio-Natiele%20Cruz-6A0DAD?style=for-the-badge&logo=github&logoColor=white)

[![Clique aqui no link](https://img.shields.io/badge/Clique%20aqui-para%20ir%20ao%20site-000000?style=for-the-badge)](https://portifolionatiele.netlify.app)


Este site funciona como uma vitrine digital, permitindo que visitantes explorem meus principais trabalhos em front-end e back-end.

## ![Sobre](https://img.shields.io/badge/O%20que%20é-Estrutura%20do%20site-000000?style=for-the-badge)

- Início: Apresenta a desenvolvedora com uma foto de perfil animada, nome e especialidade ("Desenvolvedora Fullstack").  
- Sobre: Uma seção pessoal onde Natiele compartilha sua trajetória, interesses e habilidades em programação, escrita e música.  
- Projetos: Exibe cards interativos com imagem, título, descrição e tecnologias usadas. 
- Contato: Formulário simples que envia mensagens via WhatsApp, facilitando o contato direto com a desenvolvedora.

## ![Preview](https://img.shields.io/badge/Visualiza%C3%A7%C3%A3o-Interface%20Responsiva-6A0DAD?style=for-the-badge)

 A interface é escura e elegante, com navegação fixa no topo e seções bem definidas: Início, Sobre, Projetos e Contato. A experiência é otimizada tanto para desktop quanto para dispositivos móveis.
